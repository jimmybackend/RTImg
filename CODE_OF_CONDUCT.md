# Código de conducta de RTImg

## Nuestro compromiso

Queremos que la comunidad de **RTImg** sea un espacio técnico, respetuoso y útil para quienes colaboran en diseño, implementación, documentación y revisión.

Se espera que toda interacción dentro del proyecto se mantenga profesional, especialmente cuando existan desacuerdos técnicos.

## Comportamiento esperado

- debatir ideas, no atacar personas
- discrepar con argumentos técnicos y respeto
- aceptar correcciones y revisión crítica sin hostilidad
- comunicar límites, riesgos y dudas de forma honesta
- ayudar a mantener un entorno claro para nuevos contribuidores

## Comportamiento inaceptable

No se tolerará:

- insultos o descalificaciones personales
- acoso, amenazas o intimidación
- comentarios discriminatorios
- sabotaje deliberado de discusiones o revisiones
- tergiversar el estado real del proyecto para presionar decisiones
- publicar información privada de otras personas sin permiso

## Criterio de colaboración

RTImg es un proyecto técnico en etapa temprana. Eso exige:

- discusión rigurosa
- honestidad sobre limitaciones
- respeto por la evidencia y las pruebas
- apertura a corregir decisiones de diseño

## Aplicación

Los mantenedores pueden moderar conversaciones, pedir correcciones, cerrar hilos o rechazar contribuciones que dañen el entorno del proyecto.

Cuando una interacción se desvíe, se priorizará reconducirla primero. Si la conducta persiste, podrán aplicarse restricciones de participación.

## Reportes

Si detectas una conducta problemática, repórtala a los mantenedores del proyecto con el mayor contexto posible: enlace, fecha, descripción y evidencia disponible.

## Alcance

Este código aplica a:

- issues
- pull requests
- discusiones
- revisiones de código
- documentación pública del proyecto
