# Visión general del proyecto

RTImg es el códec de imagen nativo de RT Stack. Su valor no está en competir de inmediato como formato universal de distribución, sino en resolver un problema interno real:

- normalización de formatos externos
- representación interna consistente
- transporte resiliente por tiles
- metadata operativa y trazabilidad
- base técnica compatible con RTCore, RTStream y RTCloud

En esta etapa, el proyecto se centra en un camino lossless-first con prototipo Python y especificación abierta.
